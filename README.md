# mod-automation-test
MOD - E2E automation tests for Scale Modeling project

#### Pre requirements

- R
- Python
- Nodejs

#### Install
 ```
    npm install --no-save
    npm audit
   ```


#### Run test
It is possible to run any tests throught of the BDD tags. Please, you can check this in package.json file.


```
    npm run mod:test:gui
   ```
